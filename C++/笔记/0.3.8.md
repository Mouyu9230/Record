在 `<cctype>` 中有一组**字符分类函数**，与 `isdigit`（std::isdigit）类似，用于判断字符类型。

需要头文件：

```cpp
#include <cctype>
```

## 常用函数

| 函数            | 作用          | 示例            |
| ------------- | ----------- | ------------- |
| `isdigit(c)`  | 是否为数字 `0-9` | `'3'`         |
| `isalpha(c)`  | 是否为字母       | `'a'` `'Z'`   |
| `isalnum(c)`  | 是否为字母或数字    | `'a'` `'5'`   |
| `islower(c)`  | 是否为小写字母     | `'a'`         |
| `isupper(c)`  | 是否为大写字母     | `'A'`         |
| `isspace(c)`  | 是否为空白字符     | `' '` `'\n'`  |
| `ispunct(c)`  | 是否为标点符号     | `','` `'.'`   |
| `isxdigit(c)` | 是否为16进制字符   | `0-9 A-F a-f` |
| `isprint(c)`  | 是否为可打印字符    | `'a'` `'1'`   |
| `iscntrl(c)`  | 是否为控制字符     | `'\n'`        |

## 示例

```cpp
#include <iostream>
#include <cctype>
using namespace std;

int main(){
    char c = 'A';

    cout << isalpha(c) << endl;   // 1
    cout << isdigit(c) << endl;   // 0
    cout << isupper(c) << endl;   // 1
}
```

## 字符转换函数

同一库中还有 **字符转换函数**：

| 函数           | 作用   |
| ------------ | ---- |
| `tolower(c)` | 转为小写 |
| `toupper(c)` | 转为大写 |
