C++ `std::vector` 常用操作函数分类如下：

### 1. 容量操作 (Capacity)
- `size()`: 返回当前元素个数。
- `empty()`: 判断容器是否为空。
- `capacity()`: 返回当前分配存储空间可容纳的元素总数。
- `reserve(n)`: 预留空间，避免多次重新分配。
- `resize(n)`: 改变容器大小，多余删除，不足补默认值或指定值。
- `shrink_to_fit()`: 请求减少容量以匹配大小（非强制）。

### 2. 元素访问 (Element Access)
- `operator[]`: 下标访问，不检查边界。
- `at(n)`: 下标访问，检查边界，越界抛异常。
- `front()`: 返回首元素引用。
- `back()`: 返回尾元素引用。
- `data()`: 返回底层数组指针。

### 3. 修改操作 (Modifiers)
- `push_back(val)`: 尾部插入元素。
- `pop_back()`: 删除尾部元素。
- `insert(pos, val)`: 在指定位置插入元素。
- `erase(pos)`: 删除指定位置元素。
- `clear()`: 清空所有元素。
- `emplace_back(args...)`: 原地构造尾部元素（效率优于 push_back）。
- `swap(other)`: 与另一个 vector 交换内容。

### 4. 迭代器 (Iterators)
- `begin()`, `end()`: 正向迭代器起止。
- `rbegin()`, `rend()`: 反向迭代器起止。
- `cbegin()`, `cend()`: 常量正向迭代器。
